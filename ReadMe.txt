1. The source code can be directly deployed on Windows Server 2008/2008R2/2003 and can be accessed without any issues.
2. Access denied errors won't come once deployed on II6/7/7.5
3. MVC 4 design will throw file access issues with same design.
4. Watchout for "html5shiv" & "respond.js" added in Master page to provide IE8 compatibility with bootstrap version 3.1.
5. The solution/site can be viewed on IE8 with UI responsiveness.
6. Knockout used for data binding
7. Micro ORM "Dapper" used for ORM mapping with SP's
8. Install dapper from Nuget package. On VS2010, run "Install-Package dapper" on PMC.
